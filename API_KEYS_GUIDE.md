# Get Your API Keys - Complete Guide

You need 2 FREE API keys for your weather app. Here's how to get both:

## üîë API Key 1: OpenWeather API (5 minutes)

### Step 1: Go to OpenWeather
https://openweathermap.org/api

### Step 2: Sign Up
1. Click **"Sign Up"** (top right)
2. Fill in:
   - Username: (choose any)
   - Email: your email
   - Password: (choose a strong password)
3. Check "I am 16 years old and over"
4. Check "I agree with Privacy Policy..."
5. Click **"Create Account"**

### Step 3: Verify Email
1. Check your email inbox
2. Click the verification link
3. You'll be redirected to OpenWeather

### Step 4: Get Your API Key
1. You'll see your dashboard
2. Click **"API keys"** tab
3. You'll see a default API key already created!
4. **Copy the API key** (long string of letters and numbers)
5. **Save it in a text file**

**Note:** It may take 10-15 minutes for the API key to activate. You can continue with Gemini while waiting.

## ü§ñ API Key 2: Google Gemini API (2 minutes)

### Step 1: Go to Google AI Studio
https://aistudio.google.com/app/apikey

### Step 2: Sign In
- Sign in with any Google/Gmail account
- If you don't have one, create a free Gmail account first

### Step 3: Create API Key
1. Click **"Create API Key"** button
2. Select **"Create API key in new project"**
3. Wait 5 seconds
4. Your API key appears! (starts with `AIza...`)

### Step 4: Copy Your Key
1. Click the **copy icon** next to your key
2. **Save it in a text file** with your OpenWeather key

### If You See "Upload Project"
**Don't worry!** This is optional:
- Look for **"Skip"** button and click it
- Or look for **"Continue without project"**
- Or just type any project name and click Continue

You don't actually need to upload anything!

### What You Get (FREE)
- ‚úÖ 60 requests per minute
- ‚úÖ 1,500 requests per day
- ‚úÖ No credit card required
- ‚úÖ Perfect for your weather app!

## üÜò Troubleshooting

### OpenWeather Issues

**"API key not working"**
- Wait 10-15 minutes after creation
- Check you copied the entire key
- Make sure you verified your email

**"Can't sign up"**
- Try a different email
- Check spam folder for verification email
- Use a different browser

### Gemini Issues

**"Can't find Create API Key button"**
- Make sure you're at: https://aistudio.google.com/app/apikey
- Try clicking "API Key" in the left sidebar
- Sign out and sign in again

**"Upload project screen won't go away"**
- Look for "Skip" or "X" button
- Type any project name and click Continue
- Try: https://makersuite.google.com/app/apikey (old URL)

**"API key doesn't work"**
- Make sure you copied the entire key
- Check it starts with `AIza`
- Try creating a new key
